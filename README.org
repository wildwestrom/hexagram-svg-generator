* Hexagram SVG Generator

** Run

   #+begin_src shell
   yarn install

   yarn watch
   #+end_src
** Clean

   #+begin_src shell
   yarn clean
   #+end_src
** Release

   #+begin_src shell
   yarn release
   #+end_src
